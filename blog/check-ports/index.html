Open Ports

Check if ports are open to a remote server

April 19, 2020 Dale Hassinger

1 minute read

PowerShell Code

Use PowerShell to see if ports are open to a remote server.


$PortNumber = '443'
$Destination = 'Server.vCrocs.info'

$socket = New-Object Net.Sockets.TcpClient
$socket.Connect($Destination,$PortNumber)

if($socket.Connected){
    $PortOpened = 'Port: ' + $PortNumber + ' to ' + $Destination +' is Open! :)'
    $socket.Close()
}else{
    $PortOpened = 'Port: ' + $PortNumber + ' to ' + $Destination +' IS NOT Open! :('
}
Write-Host  $PortOpened

Living in a zero trust environment can be challenging. Here is a little piece of code that I have been using to test ports from a Windows server to any type of destination. Just change PortNumber and Destination for your use case.
